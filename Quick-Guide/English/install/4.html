Step 3 - Setting File Permissions (Alternative Settings)

After you have completed the section of this guide titled "Step 2 - Uploading," you must now set the file permissions on the files listed in the table below. Since this is a Linux/*nix function, Windows server users will not be able to do this. On a Windows/NT server, however, you will need to give the data folders (those specified to be CHMOD 755) full read/write permissions. On Windows servers, you will probably need to contact your web host's support department and request that these permissions be set for you.

Return to the page with the Default CHMOD Settings.
